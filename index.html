FlexBox Row

Use these three properties to create a flexbox row layout.

.row {
display: flex;
flex-direction: row:
flex-wrap: wrap:
}
FlexBox Column

Use this to create a flexbox column layout.

.column {
display: flex;
flex-direction: row:
}
CSS Grid Layout

Build a 12 column layout to create a CSS grid

.grid {
display: grid;
width: 100%
grid-template-columns:
repeat(12, 1fr);
}
Linear Gradients

This will a create background linear gradient from top to bottom

.linear-gradient-background {
background-image: linear-gradient
rbga(232, 102, 236, 0.3) 0%,
rbga(232, 102, 236, 0.6) 100%;
}
Box Transition Glow

Use transition and box shadows to glow on hover.

.code-car .card-header {
border-radius: 8px;
transition: all 0.5s ease-in-out;
}
Overlay Card With Title

Use position properties and negative margins to raise elements higher than their natural starting point.

.card-header {
position: relative;
margin: top;
}
